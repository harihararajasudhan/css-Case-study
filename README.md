# coffee beans ☕️

A modern, clean, and fully responsive static website for a premium coffee shop. This project focuses on high-quality aesthetics, semantic HTML structure, and advanced CSS techniques.

## 🌟 Features

- **Premium Design**: Modern color palette (Deep Brown, Cream, and Burnt Orange) with a professional look and feel.
- **Responsive Layout**: Designed to look great on desktop, tablet, and mobile devices using Flexbox and CSS Grid.
- **Semantic HTML**: Fully accessible and structured using HTML5 semantic elements (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`).
- **Multiple Pages**: Includes Home, Menu, About Us, and Contact pages.
- **Interactive Elements**: Hover effects, smooth transitions, and a styled contact form.
- **Custom Assets**: High-resolution, AI-generated professional café photography.

## 🛠️ Tech Stack

- **HTML5**: For structure and content.
- **CSS3**: For styling, layout (Flexbox/Grid), and animations.
- **No Frameworks**: Built from scratch using vanilla CSS for maximum control and performance.

## 🚀 Getting Started

### Local Development
To run this project locally with a development server:

1. Clone the repository or download the files.
2. Ensure you have [Node.js](https://nodejs.org/) installed.
3. Open your terminal in the project folder.
4. Run the following command to start a local server:
   ```bash
   npm start
   ```
5. Open your browser to [http://localhost:8080](http://localhost:8080).

### Static Preview
Alternatively, you can simply open `index.html` directly in any web browser.
